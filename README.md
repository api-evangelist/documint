# Documint (documint)

Documint is a programmatic document generation platform with a template editor, REST API and integrations with major CRMs and automation tools (HubSpot, Airtable, Zapier, Make). Documents are generated by merging user-supplied JSON data into pre-built templates; data is not stored on Documint servers.

**APIs.json:** [apis.yml](apis.yml)

## APIs
- **Documint REST API** — `https://api.documint.me/v1` — templates and documents. `api_key` request header. Postman collection is canonical reference. REST API access is gated to the **Platinum** tier and above. [Docs](https://docs.documint.me/) · [Postman](https://documenter.getpostman.com/view/11741160/TVK5cLxQ).

## OpenAPI
Documint does not currently publish a downloadable OpenAPI/Swagger document at a stable public URL as of 2026-05-08; the canonical reference is the Postman collection. Pipeline did not retrieve a spec into `openapi/`.
